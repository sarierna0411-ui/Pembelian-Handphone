# AI Agent Pembelian Handphone

Aplikasi AI Agent sederhana untuk menganalisis data pembelian handphone
dan memberikan rekomendasi berdasarkan budget dan spesifikasi.

## Fitur
- Menampilkan data pembelian HP
- Rekomendasi HP berdasarkan budget
- Analisis sederhana berbasis data

## Teknologi
- Python
- Streamlit
- Pandas
